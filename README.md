# go-virbicoin GitHub Pages

This branch (`gh-pages`) hosts the landing page served at
<https://virbicoin.github.io/go-virbicoin/>.

It is a single, self-contained static page that points to the canonical
VirBiCoin resources:

- Website: <https://virbicoin.com>
- Source code: <https://github.com/virbicoin/go-virbicoin>
- Client (gvbc) documentation: <https://github.com/virbicoin/go-virbicoin/wiki>
- Protocol documentation: <https://virbicoin.com/docs>

`.nojekyll` disables Jekyll processing so `index.html` is served as-is.
