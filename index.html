go-virbicoin

The official Go implementation of the VirBiCoin protocol — an EVM-compatible, Proof-of-Work blockchain for the Metaverse economy. The client binary is gvbc.

Network at a glance

Connect tooling to VirBiCoin mainnet with these parameters.

Chain ID / Network ID 329
Native currency VBC (18 decimals)
HTTP RPC port 8329
WebSocket RPC port 8330
devp2p (P2P) port 28329
Default data directory ~/.virbicoin

Quick start

Build the client and start a node in a few commands.

# Build gvbc
git clone https://github.com/virbicoin/go-virbicoin.git
cd go-virbicoin
make gvbc

# Start a full node
./build/bin/gvbc
